# Ao Bot Commands

Welcome to the complete command reference for Ao Bot! This guide covers all user-accessible commands, their usage, and examples.

### `/profile`
<div class="command-description">
View your or another user's profile, including level, XP, rank, and achievements.
</div>

<div class="usage-section">
<h4>Usage</h4>
```bash
/profile [user] [edit]
```
</div>

<div class="options-list">
<h4>Options</h4>
<div class="option-item">
  <span class="option-name">user</span>
  <span class="option-description">The user to check profile for</span>
  <span class="option-optional">Optional</span>
</div>
<div class="option-item">
  <span class="option-name">edit</span>
  <span class="option-description">Set to "yes" to edit your bio</span>
  <span class="option-optional">Optional</span>
</div>
</div>

<div class="command-examples">
<h4>Examples</h4>
<div class="example-item">
  <code>/profile</code> - View your own profile
</div>
<div class="example-item">
  <code>/profile @username</code> - View another user's profile
</div>
<div class="example-item">
  <code>/profile edit:yes</code> - Edit your profile bio
</div>
</div>

<div class="features-list">
<h4>What it shows</h4>
<div class="feature-item">
  <span class="feature-icon">📈</span>
  <span>Current level and XP progress</span>
</div>
<div class="feature-item">
  <span class="feature-icon">🔥</span>
  <span>Daily streak information</span>
</div>
<div class="feature-item">
  <span class="feature-icon">🏆</span>
  <span>Badges and achievements</span>
</div>
<div class="feature-item">
  <span class="feature-icon">🎮</span>
  <span>Game statistics (Tic-tac-toe, Connect 4, Minesweeper)</span>
</div>
<div class="feature-item">
  <span class="feature-icon">⭐</span>
  <span>Rank and position on the server</span>
</div>
<div class="feature-item">
  <span class="feature-icon">📝</span>
  <span>Custom bio (if set)</span>
</div>

## Command Restrictions

<div class="restrictions-section">

### DM Limitations
Most commands only work within the Ao Server. Commands that work in DMs include:
- Friend request acceptance/decline buttons
- Bomb defuse buttons
- Trivia answer buttons

### Permission Requirements
Some commands require specific roles or permissions:
- **Admin Commands:** `/restart`, `/alert`, `/triggers`
- **Moderator Commands:** `/block`, `/report`
- **Developer Commands:** `/developer` (requires developer role)

</div>
