# Ao Bot Commands

Welcome to the complete command reference for Ao Bot! This guide covers all user-accessible commands, their usage, and examples.

## Command Categories

<details>
<summary>üìä **Profile & Progress Commands**</summary>

### `/profile`
**Description:** View your or another user's profile, including level, XP, rank, and achievements.

**Usage:**
```bash
/profile [user] [edit]
```

**Options:**
- `user` (optional): The user to check profile for
- `edit` (optional): Set to "yes" to edit your bio

**Examples:**
```bash
/profile                    # View your own profile
/profile @username         # View another user's profile
/profile edit:yes          # Edit your profile bio
```

**What it shows:**
- Current level and XP progress
- Daily streak information
- Badges and achievements
- Game statistics (Tic-tac-toe, Connect 4, Minesweeper)
- Rank and position on the server
- Custom bio (if set)

---

### `/leaderboard`
**Description:** View the server leaderboard sorted by different metrics.

**Usage:**
```bash

```

**Options:**
- `metrics` (optional): Sort by "level", "xp", "ai", "tictactoe", "connect4", or "minesweeper"

**Examples:**
```bash
/leaderboard              # Show XP leaderboard
/leaderboard level        # Show level leaderboard
/leaderboard tictactoe    # Show Tic-tac-toe wins leaderboard
```

### `/game`
**Description:** Create game lobbies to play with other players.

**Usage:**
```bash
/game [game_type]
```

**Game Types:**
- `tictactoe`: Create a Tic-tac-toe game lobby
- `minesweeper`: Create a Minesweeper game lobby
- `connect4`: Create a Connect 4 game lobby

**Examples:**
```bash
/game tictactoe          # Start a Tic-tac-toe game
/game connect4           # Start a Connect 4 game
/game minesweeper        # Start a Minesweeper game
```

**How it works:**
1. Use the command to create a game lobby
2. Other players can join using the generated buttons
3. Games are automatically scored and contribute to your statistics

### `/triggers`
**Description:** Manage trigger keywords and responses (Admin/Mod only).

**Usage:**
```bash
/triggers [subcommand] [options]
```

**Subcommands:**
- `list`: List all current triggers
- `add <keyword> <response> [case_sensitive]`: Add a new trigger
- `remove <keyword>`: Remove a trigger by keyword

**Examples:**
```bash
/triggers list                           # View all triggers
/triggers add hello Hi there!           # Add trigger for "hello"
/triggers remove hello                   # Remove "hello" trigger
```

## Command Restrictions

### DM Limitations
Most commands only work within the Ao Server. Commands that work in DMs include:
- Friend request acceptance/decline buttons
- Bomb defuse buttons
- Trivia answer buttons

### Permission Requirements
Some commands require specific roles or permissions:
- **Admin Commands:** `/restart`, `/alert`, `/triggers`
- **Moderator Commands:** `/block`, `/report`
- **Developer Commands:** `/developer` (requires developer role)

## Getting Help

If you need help with any command:
1. Use `/help` in the server
2. Check this documentation
3. Ask in the #help channel
4. Report bugs with `/report`

## Tips for New Users

1. **Start with `/profile`** to see your current progress
2. **Use `/daily` every day** for the XP boost
3. **Complete quests** to earn rewards and progress
4. **Make friends** to unlock gifting features
5. **Participate in games** to improve your statistics
6. **Check the catalog** for useful items to purchase
